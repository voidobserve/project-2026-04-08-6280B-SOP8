
主控暂定使用6280B-SOP8来完成
没有6280B的仿真板，只能用6280的仿真板和6280的工程来实现，最后得到的烧录文件需要用官方的`WRTConverter.exe`来转换成6280B的烧录文件

### 脚位和功能
| 引脚编号 | 名称 | 功能                                                                                             |
| -------- | ---- | ------------------------------------------------------------------------------------------------ |
| 1        | vcc  | -                                                                                                |
| 2        | P15  | 滚珠检测（外部有上拉，有震动时，检测到高电平，没有震动时，检测到低电平；建议检测时采用边沿检测） |
| 3        | P14  | 开关，控制笔头ic的供电                                                                           |
| 4        | P13  | 充电检测                                                                                         |
| 5        | P12  | LED控制（A端）                                                                                   |
| 6        | P11  | LED控制（B端）                                                                                   |
| 7        | P10  | LED控制（按键检测、LED公共端）                                                                   |
| 8        | GND  | -                                                                                                |

---

### 按键检测
检测按键时，引脚配置为输入上拉，有按键按下时，检测到低电平

### 电池电量

由于芯片本身的限制，只有以下几个挡位可以检测
| 电量挡位（单位：V） | 描述                                             |
| ------------------- | ------------------------------------------------ |
| 4.0                 | 大于表示满电，显示4格电量，小于它表示低于4格电量 |
| 3.6                 | 大于它表示3格电量，小于它表示低于3格电量         |
| 3.3                 | 大于它表示2格电量，小于它表示低于2格电量         |
| 3.2                 | 大于它表示有1格电量                              |
| 3.0                 | 小于它表示处于低电量                             |
| 2.9                 | 如果正在工作，则关机                             |

### 笔头供电
测试发现： 
| 笔头供电控制脚  | 描述            |
| --------------- | --------------- |
| 悬空            | 笔头一端 0.02 V |
| 接VDD           | 笔头一端 0.23 V |
| 接GND           | 笔头一端 1.64 V |
| 输入上拉（20K） | 笔头一端 1.64 V |
| 输入下拉（20K） | 笔头一端 4.55 V |

### 充电检测

外部100K下拉，20K上拉，上拉接5V充电口。
到引脚的电压： 5 * 100 / (100 + 20) 约为 4.16V

程序上可以直接检测引脚有没有高电平，来判断是否有充电
